# Bright-Learn-Exercises
- This repository contains a collection of exercises completed as part of Bright Learn Data Science Course.

## Aim
- To demonstrate the SQL skills gained during the Data Science Course.
- To demonstrate SQL fundamental skills in Exercises 1 - 4.
## Objectives
- Complete exercise 1: SELECT and Filtering Statements
- Complete exercise 2: Aggregate functions and Grouping
- Complete exercise 3: CASE Statement
- Complete exercise 1: Joins, UNIONS, Filtering and Aggregate

## Content
- Each exercise has 2 files: a pdf for Questions, and a pdf for Answers.
